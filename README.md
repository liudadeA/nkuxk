# 南开大学自动选课助手

[![适用平台](https://img.shields.io/badge/适用平台-浏览器油猴插件-blue)](https://www.tampermonkey.net/)
[![适配系统](https://img.shields.io/badge/适配系统-南开eamis教务系统-green)](https://eamis.nankai.edu.cn/)
[![版本](https://img.shields.io/badge/版本-v2026.03.05-orange)](https://github.com/)

专为南开大学教务选课系统（eamis.nankai.edu.cn）开发的自动化选课脚本，解决手动抢课慢、重复选课、配置繁琐等问题，适配全选课阶段。

## 🌟 核心功能
| 功能 | 说明 |
|------|------|
| 多课连选 | 支持批量添加多门课程，自动循环检测余量并抢课 |
| 定时抢课 | 自定义抢课时间，准点自动启动，重复设置自动覆盖 |
| 自动识别ID | 无需手动改代码，自动读取选课批次ID（profileId） |
| 精准防重复 | 严格校验200 OK响应，成功后冷却，杜绝反复提交 |
| 已选自动识别 | 全局扫描系统已选标记，选完自动停止 |
| 防风控设计 | 基础延迟+随机浮动，模拟真人操作节奏 |
| 可视化面板 | 可拖动面板，实时显示选课进度、ID信息 |

## 📥 安装步骤
### 1. 安装油猴插件
- Chrome/Edge/Firefox 浏览器安装 [Tampermonkey（油猴）插件](https://www.tampermonkey.net/)
- 国内用户可通过浏览器应用商店（如Edge加载项）直接安装

### 2. 导入脚本
1. 打开油猴插件面板 → 新建脚本
2. 删除默认代码，复制本项目完整脚本（见下方「脚本获取」）
3. 保存脚本（快捷键 `Ctrl+S`）

### 3. 验证安装
打开南开大学选课页面（https://eamis.nankai.edu.cn/eams/stdElectCourse.action），页面右侧会自动出现「南开选课助手」面板，即安装成功。

## 🚀 使用教程
### 基础使用
1. 打开南开教务选课页面，确保进入具体选课批次（如正选/补退选）
2. 在助手面板中：
   - 填写「课程号」：输入4位课程序号，多门用**英文逗号**分隔（如 `1234,5678`）
   - 配置参数（可选）：调整最大轮次、延迟时间（建议默认值）
   - 点击「保存配置」→「立即开始」，脚本自动抢课
3. 所有课程选完后，脚本会自动停止并弹窗提示

### 定时抢课
1. 在面板「定时开始」区域填写小时/分钟/秒（如 `08:00:00`）
2. 点击「设置/覆盖定时」，面板会显示倒计时
3. 定时时间到，脚本自动启动抢课（无需手动操作）

## ⚠️ 注意事项
1. **合规使用**：本脚本仅用于个人学习和便捷选课，请勿滥用，遵守学校教务规定
2. **延迟设置**：基础延迟建议不低于1000ms，避免请求过快被系统限流
3. **页面要求**：必须在选课页面使用，否则无法读取课程数据和ID
4. **浏览器设置**：请勿禁用页面JS，保持浏览器登录教务系统状态
5. **防重复**：脚本已内置冷却机制，无需手动干预，避免频繁点击

## ❓ 常见问题
### Q1: 提示「无法读取选课批次ID」？
- 确认当前页面是**具体选课批次页面**（不是选课首页）
- 刷新页面后重新打开助手面板
- 手动检查地址栏是否包含 `electionProfile.id=` 参数

### Q2: 脚本运行但不抢课？
- 检查课程号是否填写正确（4位数字）
- 确认课程有余量（脚本仅抢可选中的课程）
- 查看浏览器控制台（F12），排查是否有接口报错

### Q3: 面板不显示？
- 检查油猴脚本是否启用，且「匹配规则」包含南开教务域名
- 刷新选课页面（快捷键 `F5`）
- 确认脚本保存成功，无语法错误
